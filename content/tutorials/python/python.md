Python adalah bahasa pemrograman yang sangat digemari oleh kalangan coder yang baru menginjakan kakinya di dunia pemrograman, karena python memiliki struktur _syntax_ yang rapi dan mudah dipahami. Python dikategorikan sebagai bahasa tingkat tinggi yang **interpratif** dan **multiguna**.

!['Python Intro'](/post/python-intro-01.jpg "Python Intro")

Python dapat berjalan di semua _Operating System_ seperti Windows, Linux dan MacOS. Bahasa ini banyak digunakan untuk pembuatan Program CLI / GUI, _Data Analyst_, IoT , _Network Penetration_, Aplikasi Mobile dan _Artiffical Intelegence_.

<br>

## Sejarah Python 

<hr>

Mari kita kenali sejarah bahasa ini. Faktanya bahasa Python sudah ditemukan pada sekitar 1980 akhir dan berhasil Di implementasikan oleh si jenius _[Guido van Rossum](https://id.wikipedia.org/wiki/Guido_van_Rossum)_ di belanda pada bulan December tahun 1989. 

!['Pak Guido Si Penemu Python'](/post/python-intro-02.jpg "Pak Guido Si Penemu Python")

Pak Guido sendiri masih menjadi penulis utama bahasa python, walaupun source code kode python bersifat open source. Sehingga banyak sekali kontributor - kontributor untuk bahasa ini yang ikut mengembangkannya.

Singkat cerita, pada tahun 1995 Pak Guido, melanjutkan pembuatan bahasa python di sebuah organisasi perusahaan bernama ***Corporation for National Research Initative*** **(CNRI)** di Virginia, Amerika Serikat.

!['Perjalanan Para Pythonista'](/post/python-intro-04.jpg "Perjalanan Para Pythonista")

Lalu, pada bulan Mei tahun 2000, Pak Guido bersama Tim nya pindah ke organisasi perusahaan lainya yang bernama **BeOpen.com** dan membuat tim bernama BeOpen PythonLabs. 

Dan pada tahun yang sama, Setelah berhasil merilis **Python 2.0** di BeOpen.com, Tim pimpinan pak Guido ini berpindah ke **Digital Creations** yang sekarang menjadi perusahaan zope.

!['Perjalanan Para Pythonista 2'](/post/python-intro-03.jpg "Perjalanan Para Pythonista 2")

Lalu yang terakhir pada tahun 2001, dibentuklah sebuah organisasi Python yang dinamai ***Python Software Foundation** **(PSF)** yang merupakan organisasi nirlaba yang dibuat khusus untuk semua hal yang berkaitan tentang hak intelektual beserta pengembangan Python.

!['Python Foundation'](/post/python-intro-05.jpg "Python Foundation")

Dan sampai detik ini **Python Software Foundation** terus mengembangkan Python sampai ke versi **3.9** dan juga berhasil membawa pamor bahasa ini paling populer diseluruh dunia. 

Nah, *anbies* untuk yang kepo bentuk sederhana sebuah kode python, berikut adalah contoh syntax `print` pada Python :

```python
    print("Hello Sekkai")
```

Ini adalah salah satu contoh bentuk kesederhanaan _syntax_ pada python, tanpa perlu semicolon(`;`) _syntax_ dapat tereksekusi dengan baik. Sehingga dapat meminimalisir adanya _syntax error_ pada kode.
