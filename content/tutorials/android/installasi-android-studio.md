# Cara Menginstall Android Studio

## Windows

1.  Buka halaman website dari [Android Studio](https://developer.android.com/studio).
2.  Klik pilihan **Download Android Studio**.

    ![](/post/7.JPG)

3.  Akan muncul tab baru. Silahkan klik atau centang pada bagian **I have read and agree with the above terms and conditions**. Lalu klik tombol **DOWNLOAD ANDROID STUDIO FOR WINDOWS**.

    ![](/post/8.JPG)

4.  Tunggu hingga download selesai. Jika sudah selesai, buka saja file yang tadi di download.

    ![](/post/9.JPG)

5.  Silahkan **Next -> Next -> Next -> Install**. Tunggu hingga proses installasi selesai. Setelah selesai, klik **Next -> Finish**. Pastikan **Start Android Studio** pada posisi centang.

    ![](/post/10.JPG)

    ![](/post/11.JPG)

    ![](/post/12.JPG)

    ![](/post/13.JPG)

6.  Pada **Import Android Studio Settings From...**, silahkan pilih saja **Do not import settings** jika kalian tidak membackup settings Android Studio kalian sebelumnya. Setelah pilih **OK**, akan muncul box **Send usage statistics to Google**, silahkan pilih sesuai dengan keinginan Anda.

    ![](/post/16.JPG)

7.  Tekan saja **Next**, lalu setelah itu, pilih pilihan **Standard** agar Android Studio memilih settingan terbaik untuk Device Anda **-> Next**.

    ![](/post/17.JPG)

    ![](/post/18.JPG)

8.  **Select UI Theme** sesuai dengan selera Anda **-> Next**. Tunggu proses download hingga selesai

    ![](/post/19.JPG)

9.  Selamat, Anda berhasil menginstall Android Studio. Akan saya beberapa sedikit tour untuk kali ini.

    ![](/post/20.JPG)
