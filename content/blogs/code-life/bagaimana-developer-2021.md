Sekarang sudah menginjak tahun 2021, dunia sudah mulai berubah dan kita harus menyesuaikan perubahan itu. Keterampilan dan keahlian kita akan mempengaruhi segalanya, dari _problem solving_ atau pengambilan keputusan.

Karena teknologi berkembang sangat pesat, kita diharuskan untuk bisa mengetahui teknologi secara _up-to-date_, mempelajari bahasa pemrograman baru, [design patterns](https://anbidev.com/design-pattern-singleton), software beserta algoritmanya.

> **Fast Learning** dan **High Skill** developer yang bisa mengadopsi teknologi baru secara efisien selalu banyak diminati

Maka dari itu, mari kita lihat ditahun ini (2021), Bagaimana persiapan kita (Developer) dalam menghadapi _wheel-of-change_ di dunia kita.

<br>

## 1. Cloud Service Terus Berkembang

<hr>

!["Cloud Service Dikehidupan Kita"](/post/dev21-01.jpg "Cloud Service Dikehidupan Kita")

Saat ini **Cloud Computing** telah berkembang ke _"Multi-Billion Dollar"_ industri. Berdasarkan data yang diperoleh dari _Synergy Research Group_, pasar _cloud service_ telah menjadi sangat besar, dengan pendapatan lebih dari **$150 Miliar USD** hanya untuk setengah tahun pertama 2019. Pendapatan ini tumbuh sampai 24% dibandingkan tahun lalu.

Dan untuk setengah tahun pertama 2020, dikarenakan dampak COVID-19 membuat pasar industri *cloud service* mendapatkan pendapatan lebih banyak **20%** menjadi **$187 Milliar USD**. Pengeluaran diseluruh dunia untuk layanan infrastruktur cloud dan publik diperkirakan akan berlipat ganda selama lima tahun ke depan.

Pandemik COVID-19 membuat dunia menyadari pentingnya *cloud service*. Seiring meningkatnya kerja remote, kelas online dan video konferensi, cloud service menjadi bagian dalam pergerakan ekonomi dunia.

Jadi, bukan hal baru jika nanti banyak bisnis yang beralih ke bidang _cloud vendors_. Dan sejak banyak perusahaan memprediksi pergerakan cloud service, menjadikan trend ini berkembang di tahun 2021.

Kemampuan dalam mendevelop aplikasi berbasis *cloud, service* dan *product* akan menjadi salah satu skill yang dibutuhkan. Belajar tentang platform seperti AWS, Google Cloud Platform dan Microsoft Azure akan membiasakan kamu dengan *cloud environtment*.

<br>

## 2. Banyak Pasar Mengenai Big Data

<hr>

!["Perdagangan Pasar Big Data"](/post/dev21-02.jpg "Perdagangan Pasar Big Data")

Memanfaatkan potensi penuh dari **Big Data** yang dapat meningkatkan operasi bisnis, pengambilan keputusan oleh manajemen dan pemasaran produk. Sebab itulah banyak programmer mengembangkan aplikasi yang dapat mengambil keuntungan dari potensi Big Data.

**Exabytes** (Satu miliar Gigabyte) data dari kebiasaan *browsing*, menggunakan media sosial, informasi finansia dan data seluler pasti membutuhkan program agar bisa digunakan untuk proses analisis.

Banyak sekali perusahaan yang mulai bergantung pada analisis data, sehingga membuat para developer dengan pengetahuan tentang Big Data dan teknologi - teknologinya banyak dicari.

Sebagai contoh, banyak sekali kesempatan bagi para Big Data Developer di dunia trading dan saham. Perusahaan investasi ingin meningkatkan proses internal mereka dalam mengambil keputusan yang lebih baik dan meningkatkan efisiensi dalam operasional mereka.

> Trader ingin memprediksi perubahan pasar sebelum terjadi. Maka dari itu dibutuhkan mereka perlu juga memepelajari  [forecasting](https://anbidev.com/machine-learning-time-series).

<br>

## 3. Bahasa Pemrograman Rust Mulai Populer

<hr>

!["Mari Belajar Rust Bersama - Sama"](/post/dev21-03.jpg "Mari Belajar Rust Bersama - Sama")

Para programmer dan perusahaan besar seperti Microsoft, Intel dan Mozilla mulai mendukung **Rust** pada tahun 2020. Dan yang paling penting, Microsoft mempertimbangkan untuk mengadopsi bahasa ini.

Bahasa pemrograman ini banyak mendapatkan banyak perhatian dari banyak perusahaan dan ingin mengadopsi Rust kedalam sistem mereka.

Bahasa ini memungkinkan kita, menulis kode dengan performa yang lebih baik secara cepat dan aman, struktur tambahan, serta kode yang lebih cepat dan efisien.

Faktanya komunitas bahasa Rust sendiri bercita - cita membuat programming yang lebih mudan, menyenangkan dan lebih aman. Sehingga menjadikannya bahasa ini menjadi salah satu bahasa utama di dunia industri.

<br>

## 4. Investasi Kepada Developer Native App

<hr>

!["Bersemangatlah Kalian Para Developer Native App"](/post/dev21-04.jpg "Bersemangatlah Kalian Para Developer Native App")

Perkembangan industri **Mobile-App** sangatlah besar dan diperkirakan akan terus bertumbuh ditahun - tahun mendatang. Karena pasar ini adalah pasar yang menguntungkan sehingga sangat mungkin untuk berkarir menjadi *developer mobile apps*.

Tidak seperti _hybrid apps_, *native apps* adalah software yang didesain untuk berjalan di perangkat tertentu. Tentu saja, smartphone adalah salah satu yang paling populer untuk *native apps*.

Android dan iOS adalah 2 platform smartphone yang paling umum digunakan saat ini, sehingga beberapa *native apps* berfokus untuk merancang dan mendesain keduanya.

Meskipun tools seperti flutter dan React Native menawarkan *development* aplikasi yang lebih cepat, tetapi *native apps* memberikan performa yang powerfull, User-Experience yang lebih baik, dan proteksi data yang dapat diandalkan. Sehingga menjadikan *development* *native apps* akan terus berlanjut di 2021.

## Eksistensi Java Masih Relevan

<hr>

!["Java Memang Andalan"](/post/dev21-05.jpg "Java Memang Andalan")

Menurut perkiraan, **Java** akan terus mendominasi pasar pada tahun 2021. Java Developer akan selalu relevan di pasar kerja, karena java adalah salah satu bahasa pemrograman yang selalu berkembang. Namun, hal ini juga membutuhkan penyempurnaan dan peningkatan skill yang berkelanjutan oleh para java developer.

Saat ini, para java developer membutuhkan keahlian seperti android development, AWS, Docker, Git, dan Spring, bersama dengan pemahaman yang baik tentang konsep java sendiri.

Para java developer dapat dapat memahami teknologi **blockchain** dengan menjelajahi berbagai proyek *blockchain* berbasis java seperti Bitcoin, FundRequest dan Corda.

Oleh karena itu belajar java tingkat lanjut akan membuatmu tetap relevan di tahun 2021 ini.
